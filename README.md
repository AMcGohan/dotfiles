# Dotfiles

## /homefiles
* Contains `.bash_aliases` of handy commands made simplier. For example:
    * `ga` - `git add`
    * `gca "commit message"` - `git commit -m "commit message"`
    * `gp` - `git push`
* Contains a script that makes a symbolic link named `.bash_aliases` in users home directory pointing to this repositories `.bash_aliases`
