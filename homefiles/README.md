# Aliases Setup Environment

The content in this directory sets up a bunch of aliases for the users Linux shell.

* `.bash_aliases`
    * Contains a list of aliases that the user can run. These aliases were picked based on my preferences/which commands I use the most of.
    * [Link to more aliases that were borrowed for this file](https://github.com/vikaskyadav/awesome-bash-alias)
* `alias_link.sh`
    * Creates a symbolic link named `.bash_aliases` that points to the `.bash_aliases` in this directory.
    * [Creating a symbolic link](https://linuxize.com/post/how-to-create-symbolic-links-in-linux-using-the-ln-command/)
