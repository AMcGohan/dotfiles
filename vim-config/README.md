# Vim Plugin Installation

Run `vundle_install.sh` to install vundle and vim plugins

## Whats Installed?

`vundle_install.sh` will install the following plugins:

* Vundle
* vim-fugitive
* YouCompleteMe
* gruvbox
* sparkup
* Command-T
